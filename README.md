# 課題

この課題はすべて別々の関数として実装し、それぞれの関数を別々のファイルに配置してください。
課題1〜3は基本問題です。課題4は挑戦問題です。

## 課題1 `kadai1.cpp`
### 問題文

0から100までの点数を受け取り、成績をAからFの評価に変換するプログラムを作成してください。

### 入力

- 1行目に点数（整数）が与えられます。

### 出力

- 成績を1文字で出力してください（A, B, C, D, Fのいずれか）。

### 入力例

```
85
```

### 出力例

```
B
```

## 課題2 `kadai2.cpp`
### 問題文

身長（メートル）と体重（キログラム）を入力として受け取り、BMI（Body Mass Index）を計算するプログラムを作成してください。  
BMIは体重(kg) ÷ (身長(m) × 身長(m))で計算されます。

### 入力

- 1行目に身長（小数）が与えられます。
- 2行目に体重（小数）が与えられます。

### 出力

- BMIを小数点以下2桁で出力してください。

### 入力例

```
1.75
68.0
```

### 出力例

```
22.20
```

## 課題3 `kadai3.cpp`
### 問題文

西暦年を和暦に変換する関数と、和暦を西暦年に変換する関数を作成してください。  
関数オーバーロードを用いて西暦から和暦、和暦から西暦の変換を実装してください。  
ここでは簡単のため、和暦は「令和」元年を2019年とし、それ以降の年数を計算します。  
例えば、西暦2021年は和暦「令和3年」、和暦「令和5年」は西暦2023年です。

### 入力

- 入力として年号（例: 1997）を与えた場合は和暦を返す。
- 入力として和暦（例: R7）を与えた場合は西暦を返す。

### 出力

- 変換結果を出力してください。  
- 西暦→和暦の場合は「令和X年」と出力してください。  
- 和暦→西暦の場合は対応する西暦の年を出力してください。

### 入力例1

```
1997
```

### 出力例1

```
平成9年
```

### 入力例2

```
R7
```

### 出力例2

```
2025
```

## 課題4 `kadai4.cpp`
### 問題文

指定された高さのクリスマスツリーを描画するプログラムを作成してください。  
高さnのツリーは、1段目が1つの星（*）、2段目は3つの星、3段目は5つの星…と奇数個の星で中央揃えに表示します。

### 入力

- 1行目にツリーの高さn（整数）が与えられます。

### 出力

- 高さnのクリスマスツリーを描画してください。  
- 各行は空白で中央揃えされます。

### 入力例

```
4
```

### 出力例

```
   *
  ***
 *****
*******
```
